Simple demo page

This demo page shows the basics of the translate button.

Pressing the button will send a pre-authenticated request to a proxy, which will validate its autenticity and forward the request to Unbabel's REST API. The sources of this page, as well as the javascript, do not include the API key anywhere.

This text has been previously translated through Unbabel, so the previous translation is retrieved (by using the text's hash as UID).
