This demo page shows how the translate button can be generated by a server.

In this demo, the server part is done through. The server sends its authentication user and key, as well as the text to sign to the proxy web service.

The proxy web service encrypts the text using the key, and returns this token. (this could also be done on the server side with no request to the webservice, but AES encryption in javascript is awkward)

The user gets the generated page, presses the button and one of two things may happen:

  • The text was previously translated: the human translation is returned.
  • Otherwise: a machine translation is returned immediately, and the human translation is fetched on the background.
